# Professional Practices In Software Engineering

## Overview
This project is about software engineering as a profession. It explains software ethics, accreditation, licensing, professional organizations and the responsibilities of software engineers.

## Topics Covered
- ABET Accreditation
- Software Engineering Profession
- Ford & Gibbs Model
- Certification, Licensing, and Accreditation
- ACM/IEEE Code of Ethics
- BCS Code of Conduct
- IEEE and ACM Organizations
- Ethical Computing
- Environmental Impact of IT
- Pakistan Engineering Council (PEC)

## Objectives
- Learn about software engineering professionalism
- Understand ethical responsibilities
- Study licensing and accreditation systems
- Explore international professional standards
